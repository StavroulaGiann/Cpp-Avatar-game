# AvatarGame (C++)

C++ game project developed with Visual Studio as part of coursework.
The project uses a custom library provided for the course.

## Features
- Basic game loop
- Object-oriented game structure
- Input handling and gameplay logic
- Use of a custom external library

## Technologies
- C++
- Visual Studio

## Dependencies
This project depends on a custom library provided by the course instructor.
The library source code is not included in this repository.

## How to Run
1. Open `AvatarGame.sln` in Visual Studio
2. Configure the external library paths if needed
3. Build and run the project

## Notes
This repository focuses on demonstrating C++ programming,
object-oriented design and game logic rather than build configuration.
